# Releases

All official releases of the Scandium Dataset are versioned and archived.

## Release Policy

- **Semantic versioning**: major.minor.patch (e.g., v0.0.0)
- **Release candidates**: major.minor.0-rc.N before final
- **Never overwrite**: each release is immutable
- **All releases include**: data files, manifest (SHA256), statistics, validation report, release notes
