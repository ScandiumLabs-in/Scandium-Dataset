# Citation

If you use the Scandium Dataset in your research, please cite:

## BibTeX

```bibtex
@software{scandium_dataset_v3,
  title        = {Scandium Dataset v0.0},
  version      = {0.0.0},
  date         = {2026-07-21},
  publisher    = {Scandium Labs},
  doi          = {pending},
  url          = {https://github.com/Scandium-Labs/Scandium-Dataset}
}
```
