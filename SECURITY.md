# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.0.0 | ✅ |
| < 3.0 | ❌ |

## Reporting a Vulnerability

The Scandium Dataset is a scientific dataset, not a web service. Security
vulnerabilities are unlikely but if you discover a data integrity issue,
please open a GitHub issue with the "security" label.

For sensitive issues, contact the Scandium Labs maintainers directly.
